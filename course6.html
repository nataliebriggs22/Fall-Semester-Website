Course Overview

In Creating User Documentation, students learn the skills needed to effectively communicate information to diverse audiences. This class focuses on three main forms of user documentation: procedures, processes, and narrative. We are learning the best tools and practices for developing clear, user-centered content. By the end of this course, students will be able to identify, compose, and analyze different types of technical documentation, engage in audience analysis, and plan and execute effective documentation projects.
